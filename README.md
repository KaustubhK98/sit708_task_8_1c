# AI-Chatbot App

# Description

This Android app allows users to interact with Google's LLM model Gemini 2.5 Flash through a chatbot interface. The Android app communicates with the backend, and the backend sends the user's query to Gemini and returns the response.
<br><br>

## Features

- User authentication screen using username input
- Chat interface for sending and receiving messages
- AI-generated chatbot replies using Gemini 2.5 Flash
- Node.js backend integration for secure Gemini API access
- Local chat history persistence using Room Database
- Timestamp displayed on each message bubble
<br><br>

## Tech Stack

- Java
- Android Studio
- Room Database
- RecyclerView
- Node.js
- Express.js
- Gemini API
<br><br>

# Installation Instructions

Currently the app is only available on Android phones. Just install the apk from the files and run the app, no special permissions needed.
